## Why

Нужен визуальный интерфейс для демонстрации работы AI-агентов в формате пиксельного офиса — это делает абстрактные процессы наглядными и позволяет транслировать активность команды ботов в реальном времени на YouTube.

## What Changes

- Создаётся новый веб-проект (HTML/Canvas/JS) с нуля
- Пиксельная офисная сцена с зонами (рабочие столы, кулер, кухня, переговорная, принтер)
- 4 персонажа-бота с именами: CEO Алекс, Dev Макс, Accountant Лена, Manager Катя
- Каждый персонаж имеет state machine: idle → moving → working → break
- Status bubble над персонажем показывает текущую активность
- Случайное движение между зонами с анимацией ходьбы
- WebSocket-ready архитектура для подключения реальных событий в будущем

## Capabilities

### New Capabilities
- `office-scene`: Пиксельная карта офиса с зонами и объектами
- `agent-characters`: Персонажи с именами, статусами и анимацией движения
- `activity-simulation`: Симуляция случайных активностей агентов
