## ADDED Requirements

### Requirement: Office Map

Офисная сцена SHALL отрисовываться на HTML5 Canvas размером 800×600px с пиксельной графикой (tile-based, размер тайла 16px).

#### Scenario: Зоны офиса отображаются корректно

- **WHEN** страница загружается
- **THEN** canvas показывает офисные зоны: 4 рабочих стола, кулер, кухонный уголок, переговорная зона, принтер, входная зона
- **AND** каждая зона визуально различима через цвет/тайлы

#### Scenario: Объекты интерактивны визуально

- **WHEN** персонаж находится в зоне
- **THEN** зона подсвечивается или показывает индикатор активности

### Requirement: Visual Style

Графика MUST быть выполнена в пиксельном стиле (pixel art) без сглаживания.

#### Scenario: Пиксельный рендеринг

- **WHEN** canvas рендерится
- **THEN** `image-rendering: pixelated` применён к canvas элементу
- **AND** все спрайты нарисованы программно через Canvas API (без внешних изображений)
